How To Build Your Blog

Posted on 2017-02-03 |

搭建环境

  • ubuntu 16.04.4
  • Linux 4.4.0

所需工具

  • Hexo
  • Node
  • npm
  • git
  • deployer

创建项目

  1. 使用Hexo创建新的项目hexo-blog

  2. 进入工程并启动Hexo服务器

  3. 使用浏览器查看是否新建工程成功

    http://localhost:4000/

  4. 可在工程中修改全局配置文件_config.yml

  5. 写入新的博客文章

  6. 将工程进行静态化处理，并将生成的静态网页存放在public文件夹中。

  7. 提交到github上

  8. 更换主题theme，假如更换为目前正在使用的NexT主题，可直接git库，然后在hexo工程的_config.yml中的theme更改为next即可使用新的主题，在主题目录下_config.yml文档中可配置相关的参数。

  9. 如果在主题中添加tags标签，则可在hexo工程下输入

  10. 对于添加categories也是同样的操作。
