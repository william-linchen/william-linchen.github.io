常用传感器简介：加速度传感器，磁力计，陀螺仪

人拥有五官和脑，能够感知世界，相对应的，机器拥有了传感器才可以感知世界。

现在移动设备和车载终端等消费类电子设备包含越来越多的传感器，这一篇章所介绍的陀螺仪更加接近人类的小脑，负责平衡部分，而方向感的认知是加速度传感器和磁力计的合力之作。

这一章节主要介绍这三种比较容易混淆的传感器的原理，并且讲解其相同点和区别，继而延伸出各种虚拟综合类型传感器，包括重力传感器、线性加速度传感器、旋转矢量传感器等。

1. 加速度传感器

  • 介绍： 加速度传感器（也叫加速度计）是可以感知任意方向上的加速度（重力加速度则只是地表垂直方向加速度）。加速度计通过测量组件在某个轴向的受力情况来得到结果，表现形式为轴向的加速度大小和方向（XYZ）。

  • 原理： 以电容式3轴加速度计的技术原理为例。电容式加速度计能够感测不同方向的加速度或振动等运动状况。其主要为利用硅的机械性质设计出的可移动构件，构件中主要包括两组硅梳齿（Silicon Fingers），一组固定，另一组随运动物体移动；前者相当于固定的电极，后者的功能则是可移动电极。当可移动的梳齿产生了位移，就会随之产生与位移成比例电容值的改变。当运动物体出现变速运动而产生加速度时，其内部的电极位置发生变化，就会反映到电容值的变化（ΔC），该电容差值会传送给一颗接口芯片（InteRFace Chip）并由其输出电压值。因此3轴加速度传感器必然包含一个单纯的机械性MEMS传感器和一枚ASIC接口芯片两部分，前者内部有成群移动的电子，主要测量XY及Z轴的区域，后者则将电容值的变化转换为电压输出。

2. 陀螺仪

  • 介绍： 陀螺仪是一种用来感测与维持方向的装置，基于角动量不灭的理论设计出来的。其主要由一个位于轴心可以旋转的轮子构成，一旦开始旋转，由于轮子的角动量使其有抗拒方向改变的趋向。陀螺仪多用于航海和航空的导航定位系统。现消费类电子都有其应用，包括手机、游戏机等。

  • 原理： 三轴陀螺，如下图所示，其原理通过测量三维坐标系内陀螺转子的垂直轴与设备之间的夹角，并计算角速度，通过夹角和角速度来判别物体在三维空间的运动状态。三轴陀螺仪可以同时测定上、下、左、右、前、后等6个方向（合成方向同样可分解为三轴坐标），最终可判断出设备的移动轨迹和加速度。

3. 磁力计

  • 介绍： 也就是磁力传感器，而磁感应强度是矢量，具有大小和方向特征，磁力计可以测量出所处位置中地磁场和电路板器件所产生合磁场在X，Y，Z三轴中的矢量，从而计算出方位角。

  • 原理： 一般采用各向异性磁致电阻(AMR）材料来检测空间中磁感应强度的大小，这种材料对外界的磁场很敏感，磁场的强弱变化会导致AMR自身电阻值发生变化。

4. 区别与联系

从原理来说，加速度计和陀螺仪都是通过振动测量加速度的，只不过陀螺仪测量的是科氏加速度，而加速度计测量的是直接加速度。陀螺仪在结构上可以简单理解为一个谐振器+加速度计，因为陀螺仪比加速度计复杂很多，技术实现上也难得多。

陀螺仪通过测量自身的旋转状态判断出设备当前运动状态，是向前、向后、向上、向下、向左还是向右，是加速（角速度）还是减速（角速度），都可以实现，但是要判断出设备的方位（东南西北），陀螺仪就没有办法了。

磁力计是通过测量磁通量的大小来确定方向的。当磁力计发生倾斜时，通过磁力计的地磁通量将发生变化，从而使方向指向产生误差。因此，如果不带倾斜校正的电子指南针，需要用户水平放置。而利用加速度传感器可以测量倾角的这一原理，可以对电子指南针的倾斜进行补偿。
