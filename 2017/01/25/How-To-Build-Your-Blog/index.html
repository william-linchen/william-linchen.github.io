How To Build Your Blog

搭建环境

  • ubuntu 16.04.4
  • Linux 4.4.0

所需工具

  • Hexo
  • Node
  • npm
  • git
  • deployer

创建项目

  1. 使用Hexo创建新的项目hexo-blog

  2. 进入工程并启动Hexo服务器

  3. 使用浏览器查看是否新建工程成功

    http://localhost:4000/

  4. 可在工程中修改全局配置文件_config.yml

  5. 写入新的博客文章

  6. 将工程进行静态化处理

生成的静态网页存放在public文件夹中

  1. 提交到github上
